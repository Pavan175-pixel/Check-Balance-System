# Check Balance System

A simple banking application built with Python that simulates basic banking operations through a menu-driven interface.

## Features

- Check account balance
- Deposit money
- Withdraw money
- Input validation for invalid amounts
- Easy-to-use command-line interface

## Technologies Used

- Python
- VS Code

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Pavan175-pixel/Check-Balance-System-Python.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Check-Balance-System-Python
   ```

3. Run the program:
   ```bash
   python main.py
   ```

## Sample Menu

```text
1. Show Balance
2. Deposit
3. Withdraw
4. Exit
```
